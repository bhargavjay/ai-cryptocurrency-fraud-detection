# AI Cryptocurrency Fraud Detection

## 📌 Overview

This project focuses on detecting fraudulent cryptocurrency transactions using machine learning and explainable AI techniques.

## 🚀 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning models
* Federated Learning approach
* Explainable AI (SHAP & LIME)

## 🤖 Models Used

* Random Forest
* Logistic Regression
* XGBoost

## 📊 Results

* Achieved approximately 88% accuracy
* Analysed fraud transaction patterns
* Applied explainable AI to interpret model predictions

## 🧠 Tech Stack

Python • Pandas • NumPy • Scikit-learn • SHAP • LIME • Matplotlib

## 📁 Project Structure

* cryptocurrency.ipynb → Main notebook
* Cryptocurrency_Transaction_Data.csv → Dataset

## 🔍 Key Learnings

* Handling imbalanced datasets
* Model interpretability
* Feature importance analysis

## 🚀 Future Improvements

* Improve fraud class recall
